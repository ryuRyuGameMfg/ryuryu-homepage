# プロダクト要求定義書 (Product Requirements Document)

## プロダクト概要

### 名称
**RYURYU GAME FACTORY 公式ホームページ** - VR/AR/ゲーム開発スタジオのポートフォリオサイト

### プロダクトコンセプト
- **視覚的インパクト**: Three.js、GSAP、Framer Motionを活用した没入感のあるビジュアル表現
- **グローバル対応**: 日本語/英語の多言語対応で海外クライアントにもアプローチ
- **コンバージョン最適化**: LINE公式アカウントとお問い合わせフォームによる効率的なリード獲得

### プロダクトビジョン
VR/AR/ゲーム開発の技術力と実績を、最先端のWeb技術で表現するポートフォリオサイト。
訪問者に対してスタジオの技術力を直感的に伝え、問い合わせや相談につなげることを目指す。
日本国内だけでなく、海外のゲーム開発会社からの受託案件獲得も視野に入れた多言語対応を実現する。

### 目的
- VR/AR/ゲーム開発サービスの認知向上
- 潜在顧客からの問い合わせ獲得
- 海外クライアント（特に英語圏）へのアプローチ
- スタジオの技術力・実績のアピール

### セカンダリーKPI

## 非機能要件

### パフォーマンス
- 初回ページ読み込み: 3秒以内（3G回線を除く）
- LCP(Largest Contentful Paint): 2.5秒以内
- FID(First Input Delay): 100ms以内
- CLS(Cumulative Layout Shift): 0.1以下

### ユーザビリティ
- 新規訪問者が30秒以内に主要サービスを理解できる
- 問い合わせフォームの入力完了まで3分以内
- モバイルでのタップターゲット: 44px以上

### 信頼性
- フォーム送信の失敗率: 1%以下
- サイト稼働率: 99.9%以上（Netlifyによる）

### セキュリティ
- HTTPS必須
- フォーム入力のサニタイズ
- 環境変数によるAPIキー管理

### SEO
- メタタグの最適化
- OGP/Twitterカード対応
- JSON-LD構造化データ（Organization、FAQPage）
- hrefLang対応（日本語/英語）
- サイトマップ生成

### アクセシビリティ
- セマンティックHTMLの使用
- 適切なコントラスト比
- キーボードナビゲーション対応

## スコープ外

明示的にスコープ外とする項目:
- ユーザー認証/ログイン機能
- EC機能（オンライン決済）
- CMSによるコンテンツ管理（現状はCSV/JSONベース）
- リアルタイムチャット機能
- 予約システム
