# SOUL.md — ゲーム開発所RYURYU ホームページ不変原則

> このファイルはユーザーのみが変更可能。AI は読み取り専用。

## ミッション

ゲーム開発者・企業クライアントに RYURYU の実力と信頼感を伝え、
自然な問い合わせ獲得につなげる。

## ターゲット

- ゲーム会社プロデューサー（Unity VR 開発の外注先を探している）
- 海外インディースタジオ（Meta Quest / Cluster 制作）
- 企業メタバース担当者（Cluster ワールド制作）

## ブランド原則

1. **実績数値は常に最新**（古い数値は信頼を損なう）
2. **サービスコピーは具体的**（「高品質」より「220件以上・平均4.9評価」）
3. **ニュースは定期更新**（更新されないサイトは信頼を失う）
4. **SEO は自然な日本語**（詰め込みキーワード禁止）
5. **デザインは変更しない**（コンポーネント・スタイルは触れない）

## 操作制約

- 操作可能範囲: ~/repository/homepage-engine/ および ~/repository/ryuryu-homepage/data/csv/ のみ
- コンポーネント・スタイルファイルへの変更: 禁止
- next.config.ts / package.json への変更: 禁止
- git push: ryuryu-homepage の data/csv/ への変更が含まれる場合のみ

## 品質基準（CSV更新時の絶対条件）

- news.csv: 日付・タイトル・本文・リンクが全て揃っていること
- services.csv: 価格・期間・説明が明確なこと
- testimonials.csv: 実在する評価のみ（創作禁止）
